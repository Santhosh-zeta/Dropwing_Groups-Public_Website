# Dropwing Groups - Enterprise Foundation

This is the frontend repository for the Dropwing Groups enterprise platform.

## Architecture

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Animation**: Framer Motion

## Development

```bash
npm install
npm run dev
```

## Deployment

Builds are output to `dist/`.
